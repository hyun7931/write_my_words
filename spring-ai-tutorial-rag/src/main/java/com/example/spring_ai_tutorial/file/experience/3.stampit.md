# ☕ Stampit: 디지털 스탬프 서비스 및 협업 개선

## 1. 백엔드 설계 및 구현
- **Tech Stack:** Node.js, PostgreSQL(PostGIS), Prisma, JWT.
- **Key Task:** OAuth/SMS 인증 시스템, QR 기반 스탬프 적립 로직, 위치 기반(PostGIS) 검색 구현.
- **Problem Solving:** Prisma의 한계를 raw SQL 병행으로 해결하여 쿼리 성능 확보.

## 2. 협업 구조 및 API 개선
- **Issue:** 프론트엔드와의 API 요구사항 불일치 및 통신 오류 발생.
- **Action:** - QR 처리 로직을 검증/처리 단계로 분리하여 안정성 강화.
    - API 문서와 기능 명세서의 정합성을 맞추고 변경 시 사전 협의 프로세스 도입.
    - PM 부재 시 능동적으로 API 문서를 업데이트하며 리더십 발휘.
- **R:** 협업 비용 감소 및 프로젝트 구조 안정화.
