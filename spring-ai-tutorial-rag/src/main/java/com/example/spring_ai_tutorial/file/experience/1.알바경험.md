## 1. 빵집 아르바이트 (운영 안정화)
- **S:** 주말 마감 담당으로 근무하며 2일간의 짧은 인수인계 후 단독 운영 상황 직면.
- **T:** 실수 없는 마감 업무 수행 및 매장 운영 안정성 확보.
- **A:** - 인수인계 내용을 체크리스트로 구조화하여 업무 누락 방지.
    - 마감 순서를 표준화하고 피드백을 반영하여 지속적으로 매뉴얼 개선.
    - 신규 근무자를 위한 인수인계 문서 제작 및 교육 진행.
- **R:** 빠른 독립 업무 수행으로 점주로부터 높은 신뢰 확보.

## 2. 학원 데스크 아르바이트 (시스템 개선)
- **S:** 간략한 인수인계로 인한 업무 혼선 및 후임자의 적응 어려움 예상.
- **T:** 업무 효율화 및 후임자 적응 환경 구축.
- **A:** - 발생 빈도가 높은 질문과 해결 방법을 사진과 설명이 포함된 문서로 제작.
    - 결제 구조 원리를 파악하여 결제 실수 방지 프로세스 구축.
- **R:** 원장님이 해당 문서를 공식 인수인계 자료로 채택, 업무 오류 현저히 감소.
