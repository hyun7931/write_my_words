# 🤖 DeepTicker: AI 기반 맞춤형 뉴스 해설 서비스

- **S:** 경제 뉴스 요약 및 사용자 수준별 맞춤 해설 제공 AI 서비스 개발.
- **T:** 사용자 수준에 맞는 뉴스 이해 지원 시스템 구현.
- **A:** - **데이터 모델링:** 뉴스 크롤링 및 카테고리 분류 모델 구축.
    - **AI 엔지니어링:** Gemini 기반 요약 및 프롬프트 엔지니어링 적용, NER 기반 용어 추출 DB 연동.
    - **품질 관리:** ROUGE, BLEU 기반 품질 평가 및 재요약 로직 구현.
    - **전략적 판단:** 복잡도에 따른 기능 우선순위 조정으로 핵심 기능 우선 구현.
- **R:** 교내 공모전 장려상 수상, 수준별 맞춤형 서비스 구현 성공.
